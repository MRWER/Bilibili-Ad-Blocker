# Bilibili-Ad-Blocker - B站评论区广告一键清剿

一个 Chrome 扩展，用于自动识别并清理 B 站视频评论区中的广告账号。通过关键词评分、用户画像分析、空间二次验证以及异常纯净画像识别，精准定位可疑评论，并提供一键拉黑或自动批量清剿功能。

## ✨ 主要功能

- **智能检测**  
  结合关键词正则、用户等级、头像特征、用户自定义词库等多维评分，精准识别广告/引流评论。

- **空间二次验证**  
  对触发了关键词阈值的低等级账号，自动检查其空间签名、最新动态、动态评论区以及视频标题中是否含有引流词，降低误伤。

- **异常纯净画像识别**  
  针对粉丝数、获赞数、播放数极低（<10）或动态全为纯转发的“三无”账号，直接判定为广告号。

- **一键清剿**  
  在检测到的可疑评论后添加“🚫 清剿”按钮，点击即可一键拉黑（≤3 级账号直接拉黑，≥4 级仅提示手动举报）。

- **自动清剿队列**  
  勾选 Popup 中的自动清剿后，所有符合条件的低等级广告号将自动加入队列，每秒处理一个，并在右上角面板实时显示进度。

- **自定义词库 + 标记学习**  
  每条评论旁都有一个“📌 标记”按钮，点击即可将该评论的特征词加入永久词库，后续刷新生效。也可在 Popup 中手动编辑词库。

- **高等级账号保护**  
  对于 ≥4 级账号或被识别为“硬核会员”的账号，仅弹窗提示，不执行拉黑，防止误伤被盗高级号。

- **Shadow DOM 穿透监听**  
  适配 B 站新版 Shadow DOM 评论区，支持懒加载和楼中楼，实时扫描新出现的评论。

## 🚀 安装与使用

1. 下载本仓库所有文件到本地文件夹。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 勾选右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择项目文件夹。
5. 确认插件已安装并启用。
6. 打开任意 B 站视频页面（`https://www.bilibili.com/video/...`），滚动到评论区。
7. 可疑评论右侧会出现“🚫 清剿”按钮和“📌 标记”按钮。
8. 点击插件图标，可在弹出窗口中：
   - 查看当前页面状态
   - 开启/关闭自动清剿
   - 编辑自定义词库

## ⚠️ 风险提示

本插件仅供学习与研究目的使用。自动化拉黑操作可能违反 B 站服务条款，存在账号被限制或封禁的风险。请使用小号测试，谨慎操作。插件所有数据仅存储在浏览器本地，不会上传到任何服务器。

## 🧪 技术要点

- **WBI 签名**：实现了符合 B 站最新规范的 WBI 签名算法（使用预定义映射表生成 `mixin_key`），用于视频列表等需要认证的接口。
- **请求队列**：为防止频率限制，API 请求通过队列管理，间隔 1.5 秒，并自动重试。
- **空间信息缓存**：同一 UID 的空间检测结果缓存 5 分钟，避免重复请求。
- **隐私与安全**：Cookie 等敏感信息直接从浏览器读取，不要求用户手动输入；所有数据存储在 `chrome.storage.local`。
