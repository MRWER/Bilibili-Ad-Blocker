# Bilibili-Ad-Blocker - B站评论区广告一键清剿

一个 Chrome 扩展，用于自动识别并清理 B 站视频评论区中的广告账号。通过**规则引擎 + 增量朴素贝叶斯分类器**的混合模型，结合空间二次验证、异常纯净画像识别，精准定位可疑评论，并提供一键拉黑或自动批量清剿功能。

## ✨ 主要功能

- **智能检测**  
  融合关键词正则、用户等级、头像特征等规则评分，以及**在线朴素贝叶斯模型**的语义概率，精准识别广告/引流评论。

- **机器学习 + 自定义词库**  
  每条评论旁的“📌 标记”按钮会同时将特征词加入**用户词库**和**贝叶斯模型**，双重学习。您也可以在弹出窗口中手动编辑词库。模型完全本地运行，词库与贝叶斯协同工作，无需手动干预。

- **空间二次验证（增强版）**  
  对触发了分数阈值的低等级账号，自动检查其空间签名、最新动态、动态评论区及视频标题是否含有引流词。新增：
  - 优化 `hasAdKeywords` 函数，支持收费暗示（“不免费3698768584”）、索取暗示（“要就给”）、情感暗示（“想要你陪我”）、`by` 标题引流、“群：数字”等模式。
  - 擦边视频标题检测（如“扭扭臀放松一下”）+ 空签名组合判定。
  - 签名与视频标题组合暗示检测（如“白天有课”+“不想用八级甲了”）。

- **异常纯净画像识别**  
  针对粉丝数、获赞数、视频数均极低（<10）或动态全为纯转发的“三无”账号，直接判定为广告号。

- **一键清剿**  
  在检测到的可疑评论后添加“🚫 清剿”按钮，点击即可一键拉黑（≤3 级账号直接拉黑，≥4 级仅提示手动举报）。

- **自动清剿队列**  
  勾选 Popup 中的自动清剿后，所有符合条件的低等级广告号将自动加入队列，每秒处理一个，并在右上角面板实时显示进度。

- **高等级账号保护**  
  对于 ≥4 级账号或被识别为“硬核会员”的账号，仅弹窗提示，不执行拉黑，防止误伤被盗高级号。

- **Shadow DOM 穿透监听**  
  适配 B 站新版 Shadow DOM 评论区，支持懒加载和楼中楼，实时扫描新出现的评论。

## 🚀 安装与使用

1. 下载本仓库所有文件到本地文件夹。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择项目文件夹。
5. 确认插件已安装并启用。
6. 打开任意 B 站视频页面（`https://www.bilibili.com/video/...`），滚动到评论区。
7. 可疑评论右侧会出现“🚫 清剿”按钮和“📌 标记”按钮。
8. 点击插件图标，可在弹出窗口中：
   - 查看当前页面状态
   - 开启/关闭自动清剿
   - **编辑自定义词库**（手动添加/删除关键词）
   - **重置贝叶斯模型**（清空所有已学习的样本）

> **提示**：您标记的评论会同时学习到词库和贝叶斯模型，两者互补，提升识别准确率。

## ⚠️ 风险提示

本插件仅供学习与研究目的使用。自动化拉黑操作可能违反 B 站服务条款，存在账号被限制或封禁的风险。请使用小号测试，谨慎操作。插件所有数据仅存储在浏览器本地，不会上传到任何服务器。

## 🧪 技术要点

- **混合评分模型**：规则分（0~100）与贝叶斯概率（0~1）加权融合（默认 0.6:0.4），最终分数 ≥40 触发清剿。
- **增量朴素贝叶斯**：特征哈希（10,000 维）、拉普拉斯平滑、Map 存储计数，支持增量更新，完全本地化。
- **自动样本平衡**：用户标记广告为正样本，空间验证正常的评论延迟 5 秒作为负样本，防止模型偏向。
- **增强空间检测**：复用评论规则引擎，对签名、动态、视频标题进行评分，阈值 30；新增擦边标题、组合暗示检测。
- **WBI 签名**：实现了符合 B 站最新规范的 WBI 签名算法，用于视频列表等需要认证的接口。
- **请求队列**：API 请求通过队列管理，间隔 2.2 秒，自动重试被频率限制的请求。
- **空间信息缓存**：同一 UID 的空间检测结果缓存 5 分钟，避免重复请求。
- **隐私与安全**：Cookie 等敏感信息直接从浏览器读取，所有数据存储在 `chrome.storage.local`。
